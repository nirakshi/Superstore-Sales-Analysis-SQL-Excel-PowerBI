# 📊 Superstore Sales Analysis & Business Intelligence Dashboard

## 📌 Project Overview

This project analyzes the Superstore Sales dataset using SQL, Microsoft Excel, and Power BI. The objective is to extract meaningful business insights, calculate key performance indicators (KPIs), and build an interactive dashboard for decision-making.

---

## 🛠 Tools Used

- Microsoft SQL Server
- Microsoft Excel
- Power BI
- Git & GitHub

---

## 📂 Project Workflow

1. Data Import
2. SQL Data Analysis
3. Excel Pivot Analysis
4. Power BI Dashboard
5. Business Performance Report

- Monthly Sales Trend
- Sales by Customer Segment
- Top 10 Customers
- Interactive Slicers

---

## 📑 Business Insights

- West region generated the highest sales.
- Technology category produced the highest profit.
- Consumer segment contributed the highest revenue.
- Higher discounts reduced profitability.
- Monthly sales showed seasonal trends.
